Scheduled Daily Standup Report

Work Automation
Beginner

Automatically open Slack, navigate to a specific channel, and type a standup message every morning at 9 AM.

What it does:

  • Launches Slack (or activates if already open)
  • Searches for the #daily-standup channel
  • Types a pre-formatted standup template
  • Presses Enter to post

Setup:

  • Create a new macro named "Daily Standup"
  • Set hotkey to ‚åò + ‚å• + S
  • Record: Open Slack ‚Üí Search channel ‚Üí Type message
  • Use the Scheduler to run at 9:00 AM every weekday

Content Creation

Email Signature & Template Auto-fill

Email Automation
Intermediate

Quickly create emails with pre-formatted signatures and templates.

What it does:

  • Opens Mail app (or Gmail in browser)
  • Clicks "New Message"
  • Types a professional signature block
  • Positions cursor for recipient field

Tips:

  • Create separate macros for different email types (Follow-up, Complaint, etc.)
  • Use delays to wait for app responses
  • Add multiple versions with different templates

Backup Files to Cloud Storage

System Automation
Intermediate

Automatically backup important files to iCloud, Google Drive, or Dropbox.

What it does:

  • Opens Finder
  • Navigates to specific folders
  • Copies files to cloud storage folder
  • Logs completion to a notification

Best practices:

  • Schedule to run daily at specific times
  • Add error handling (wait for operations to complete)
  • Consider file size to avoid overloading network
