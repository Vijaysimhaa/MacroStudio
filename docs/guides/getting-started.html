System Requirements

  • macOS 13.0 or later (Ventura or newer)
  • Apple Silicon (M1/M2/M3) or Intel Mac
  • 5.6 MB disk space
  • Accessibility permissions (granted on first launch)
üí° Tip: Make sure to grant MacroStudio Accessibility permissions in System Preferences for full functionality.

Installation

1Download

Download the latest MacroStudio DMG file from the home page.

2Mount

Double-click the DMG file to mount it. You'll see the MacroStudio icon and the Applications folder.

3Install

Drag the MacroStudio app to the Applications folder.

4Launch

Open Applications folder and double-click MacroStudio to launch.

5Grant Permissions

MacroStudio will ask for Accessibility permissions. Click "Open System Preferences" and enable MacroStudio under Security & Privacy > Accessibility.

Creating Your First Macro

Step 1: Open MacroStudio

Launch the app from your Applications folder. You'll see the main window with the macro list.

Step 2: Create a New Macro

Click the "+" button to create a new macro. Give it a descriptive name like "Copy & Format" or "Send Email".

Step 3: Set a Hotkey

Click on the hotkey field and press your desired keyboard shortcut. For example: ‚åò + ‚å• + C (Command + Option + C)

üí° Tip: Avoid using hotkeys that conflict with system shortcuts. MacroStudio will warn you if there's a conflict.

Step 4: Record Your Actions

Click the "Record" button and perform the actions you want to automate:

  • Type text
  • Click buttons
  • Move the mouse
  • Press keyboard shortcuts
  • Switch between apps

Step 5: Stop Recording

When done, click "Stop" to finish recording. MacroStudio captures every action with millisecond precision.

Step 6: Test & Edit

Press your hotkey to run the macro. If something isn't right, use the Visual Editor to tweak timing or adjust specific actions.

Step 7: Save

Click "Save" to store your macro. It's now ready to use anytime!

Pro Tips

  • Add Delays: Click between actions to add waiting time (useful for app loading)
  • Edit Events: Double-click any action in the timeline to modify it
  • Duplicate Macros: Right-click a macro to duplicate and modify it
  • Schedule Macros: Set a macro to run at specific times with the Scheduler
  • Mouse Precision: Use "Wait for Element" to click the same spot reliably
