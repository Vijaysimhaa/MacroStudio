General Questions

What is MacroStudio?
MacroStudio is a native macOS automation tool that lets you record and playback keyboard shortcuts and mouse actions. It's perfect for automating repetitive tasks, creating workflows, and boosting productivity. Built entirely in Swift with SwiftUI for maximum performance and reliability.
Is MacroStudio free?
Yes! MacroStudio is free to download and use. It's soon to be open sourced, so everyone can benefit from it. We believe automation should be accessible to everyone.
How is MacroStudio different from Keyboard Maestro?
While Keyboard Maestro is more feature-rich and programmable, MacroStudio focuses on simplicity and visual recording. Key differences:

MacroStudio:
‚úì Visual recording and editing
‚úì Simple, intuitive UI
‚úì Lightweight (8MB vs Keyboard Maestro's 40MB+)
‚úì Free and open source

Keyboard Maestro:
‚úì More advanced scripting
‚úì Conditional logic and loops
‚úì More customization options
‚úì Paid ($36)
Will MacroStudio work on my Mac?
MacroStudio requires macOS 13.0 (Ventura) or later and works on both Apple Silicon (M1/M2/M3) and Intel Macs. Check your macOS version: Apple menu ‚Üí About This Mac ‚Üí macOS version.
Can I use MacroStudio on multiple Macs?
Yes! You can install MacroStudio on multiple Macs. Each installation is independent, so you'll need to create/export macros for each device separately. Export functionality is coming in a future update.

Features & Capabilities

Can MacroStudio record my mouse clicks and keyboard input?
Yes! MacroStudio records both mouse movements, clicks, and keyboard input with millisecond precision. You can record any action you can manually perform, then play it back instantly with a hotkey.
Can I edit macros after recording?
Absolutely! MacroStudio includes a Visual Editor where you can:
‚úì Reorder actions
‚úì Adjust delays between actions
‚úì Modify click positions
‚úì Edit typed text
‚úì Add or remove individual actions
Does MacroStudio support scheduling?
Yes! You can schedule macros to run at specific times or on recurring schedules. Perfect for automating daily standups, weekly reports, or any time-based task. Note: MacroStudio must be running for scheduled macros to execute.
Can I create conditional macros (if/then logic)?
The current version focuses on linear recording and playback. Advanced conditional logic and scripting are planned for future versions. For now, you can create separate macros for different scenarios.
How many macros can I create?
There's no limit! Create as many macros as you need. We have users with 100+ macros for different tasks and workflows. They're all easily accessible from the main window.

Privacy & Security

Is MacroStudio safe and secure?
Yes! MacroStudio is completely safe:
‚úì Built with native macOS security features
‚úì All data stays on your Mac (no cloud services)
‚úì No telemetry or tracking
‚úì No ads or third-party integrations

Does MacroStudio send my data to the cloud?

Optional (recommended):
‚úì Full Disk Access - Allows macros to work with files on your entire Mac

We don't recommend storing passwords in macros. Instead:
‚úì Use macOS Keychain with auto-fill features

Technical Questions

How accurate is the recording?
MacroStudio records with millisecond precision. Every click position, keystroke, and delay is captured accurately. This ensures reliable playback, especially for time-sensitive actions.
What happens if my screen configuration changes?
If you change your display setup (e.g., connecting to an external monitor or changing resolution), recorded click positions might be off. Solution:
‚úì Re-record macros on your new setup
‚úì Or manually adjust click coordinates in the Visual Editor
Can MacroStudio run macros in the background?
Yes! Macros can run even when MacroStudio isn't the active window. Just keep it open (minimize to dock). The app will work behind other windows, browsers, and applications.
How much disk space does MacroStudio use?
The app is only 8MB to download and install. Your macros are stored locally and use minimal space (typically a few KB per macro). Even with 100+ macros, storage usage remains negligible.
Does MacroStudio work with web apps?
Yes! MacroStudio works with any app on your Mac:
‚úì Web browsers (Chrome, Safari, Firefox)
‚úì Web apps (Gmail, Slack, GitHub, etc.)
‚úì Native apps
‚úì Terminal and command-line tools

The only limitation: you can't control mouse/keyboard in full-screen games (they block system-level input).

Usage & Workflow

Can I export or backup my macros?
Export/import functionality is coming in a future update. For now, macros are stored at: ~/Library/Application Support/MacroStudio/
What's the longest macro I can create?
There's no practical limit! MacroStudio can handle macros with thousands of actions. Very long macros (1000+ actions) might take a moment to load in the editor, but they'll run fine.
Can I pause a running macro?
Once a macro starts executing, it runs to completion. However, you can press the hotkey again to start a different macro, or wait for the current one to finish. Pause functionality is planned for future versions.
Can I run multiple macros simultaneously?
Currently, macros run one at a time. If you trigger a second macro while one is running, it will queue and start after the first completes. Parallel execution is being explored for future versions.
How do I organize macros into folders or groups?
Macro organization features (folders, tags, search) are planned for future updates. For now, use descriptive naming conventions (e.g., "Work_DailyStandup", "Email_Template_Follow-up") to keep macros organized.
